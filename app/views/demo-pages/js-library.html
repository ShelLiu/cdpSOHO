Javascript Library

说明：

公用方法分为两类：

  1. 以 _ 为命名空间的，属于 Underscore 的扩展方法，为基础工具，不涉及 DOM 操作。详见 basic-tools.js。
  2. 以 CDP 为命名空间的，是比较复杂的 DOM 元素操作，object 处理等等。详见 underscore-extension.js。

CDP.setLang

CDP.setLang():          // set default language to the whole page
CDP.setLang(lang):      // set target language to the whole page
CDP.setLang($el):       // set default language to certain $el
CDP.setLang(lang, $el): // set target language to certain $el
