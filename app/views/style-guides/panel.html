代码：

<div class="panel">
  <div class="header">
    <h4>面板标题</h4>

    <div class="opr">
      <button type="button" class="btn-link">×</button>
    </div>
  </div>

  <div class="content">...</div>
</div>

说明：

  • .panel 样式全站通用，目前 .dialog、.widget 都使用这套格式。
  • 要使用 .panel 样式，必须按照如下格式组织元素结构：
    .panel
  - .header  // 头部
    - h4     // 标题
    - .opr   // 操作按钮
  - .content // 内容
  • 在白底色上看，面板的边框不清晰。实际使用时下方会有图片或深色背景。
