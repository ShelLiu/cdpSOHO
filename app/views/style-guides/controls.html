按钮

显示效果：

代码：

<button type="button">普通的按钮</button>
<button type="button" class="btn-primary">牛逼的按钮</button>
<button type="button" class="btn-secondary">内涵的按钮</button>
<button type="button" class="btn-gray">低调的按钮</button>
<button type="button" class="btn-light-gray">忧郁的按钮</button>
<button type="button" class="btn-white">宛若新生的按钮</button>
<button type="button" class="btn-link">白白的按钮</button>
<button type="button" class="btn-link btn-link-primary">不太白的按钮</button>

说明：

  • 按钮统一使用 button 标签，禁用 <input type=button> 和不符规范的 a，避免不同浏览器的渲染差异。
  • 普通型：button。
  • 高亮型：button.btn-primary。
  • 链接型：button.btn-link。
  • 链接高亮型：button.btn-link-primary。
  • 禁用方式：<button disabled>，不要写 disabled=true、disabled=disabled，避免混乱。

隐藏按钮

显示效果：

代码：

<div class="btn-group">
  <button type="button">剪切</button>
  <button type="button">复制</button>

  <div class="btn-overflow">
    <button type="button" class="btn-icon btn-trigger">┇</button>

    <div class="btn-stack">
      <button type="button">粘贴</button>
      <button type="button">粘贴为纯文本</button>
    </div>
  </div>
</div>

说明：

  • 在收起状态下，点击 .btn-trigger，展开 .btn-stack。
  • 在展开状态下，点击 .btn-trigger，.btn-stack 上的按钮，和页面其他部位都能收起 .btn-stack。
  • .btn-stack 展开/收起方式和其他下拉列表通用。

checkbox

显示效果：

代码：

<label class="check"><input type="checkbox" checked> <span>选择项 1</span></label>
<label class="check"><input type="checkbox" disabled> <span>禁用</span></label>
<label class="check"><input type="radio" checked> <span>选择项 1</span></label>
<label class="check"><input type="radio" disabled> <span>禁用</span>

说明：

  • 选择项每条都用 label.check 打包，文字部分用 span 打包，否则无法显示出样式。
  • span 开头有一个空格。
  • checked 属性不要写成 checked=true 或 checked=checked。
